### 浙北党员之家助手（DyzjAssistant v2.0）使用手册

本插件专为提升基层党务工作效率设计

目前已集成：**支部主题党日活动核查、批量签到**两大功能



![Snipaste_2026-01-19_12-25-55.png](https://bexp.135editor.com/files/users/1587/15872326/202601/Eeaf9vIn_VxJF.png?auth_key=1769356799-0-0-3ed5211c2c2f770d0b7913e35fe0ee15)



**解决的问题：**

①每月的支部是否建立了活动，活动是否包含【主题党日】活动类型，需要单个支部逐个点开检索，效率很低

②党员活动签到，参加活动了但未签到，或者身份证刷卡机损坏导致整个支部未录入，需用【二维码批量补签】等情况



![img](https://q9.itc.cn/images01/20250813/a12fe0fe00c94951a351fca8777372e5.gif)

**使用方法：**

**①【活动核查】功能**

**前提：要在【活动管理】页面才能输出正常结果！**

**![img](https://bexp.135editor.com/files/users/1587/15872326/202601/TYyXubQD_pPF8.png?auth_key=1769356799-0-0-66697ac9111aaf991ce679dcdd0d0a07&x-bce-process=image/auto-orient,o_1)**

输入要查询的年月，如2025年12月，点击开始核查

![Snipaste_2026-01-19_14-07-55.png](https://bexp.135editor.com/files/users/1587/15872326/202601/VFp2jmrF_HuPK.png?auth_key=1769356799-0-0-1d1007d78a079014151a9138004aa687)

耐心等待核查完成之后，可在插件中查看核查结果，可以更方便的根据核查结果进行改正，目前总共有三种状态：

**达标：开了会，并且有【主题党日】类型**

**缺少主题党日：开了会，但没有【主题党日】类型**

**本月无活动：该支部这个月没有开展主题党日活动**

**②【批量签到】功能**

帮未带手机或操作不便的党员集体补签，无需一个一个扫码录入。

（1）从【长兴先锋】公众号中取出活动补签二维码后，保存图片

![Snipaste_2026-01-19_15-01-09.png](https://bexp.135editor.com/files/users/1587/15872326/202601/wZ8II8VN_Daz7.png?auth_key=1769356799-0-0-47ba20bb8cb52df58b0e9ff7a27fde32)

（2）将图片文件拖拽进插件虚线框内，看到“解析成功！活动ID：XXXXXX”即可

![img](https://bexp.135editor.com/files/users/1587/15872326/202601/7yxjOtfY_k7N5.png?auth_key=1769356799-0-0-61a977ca4a8f7b307aadae916cd5d79f)

（3）准备签到人员名单，在word中整理两列数据，**身份证号+入党日期**（年月六位数），中间用空格隔开，格式示例如下：



身份证号码1 200907

身份证号码2 200908

身份证号码3 200910

...

（4）将两串数字复制粘贴到插件文本框中

![Snipaste_2026-01-19_15-05-28.png](https://bexp.135editor.com/files/users/1587/15872326/202601/qRgOjEOz_sr4s.png?auth_key=1769356799-0-0-1b79f812a637ad6039a7867e2eca9d76)

（5）点击【开始批量签到】即可,等待完成

![img](https://bexp.135editor.com/files/users/1587/15872326/202601/sRSJ7aKs_qu2m.png?auth_key=1769356799-0-0-6c112889c78daaa788188b35a94a21df)

**常见问题（Q&A）**

Q：为什么点击【活动核查】之后出来的数据是错的或者没反应？

**A：请确保当前处于“活动管理”页面下**

Q：扫码解析失败怎么办？

**A：确保截图清晰，二维码要全，并且无遮挡**

Q：批量签到安全吗？

**A：本工具完全在本地运行，不会将任何人员信息上传到网络。签到频率模拟了人工操作，完全安全。**
